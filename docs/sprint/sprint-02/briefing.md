### このスプリントでやること

#### A. Sprint 1 持ち越し修正（先に対応する）

| # | 対応内容 | 参照先 |
|---|---------|--------|
| A-1 | FAQ に5問目「相談だけでもできますか？」を追加 | `lp-content.md` セクション3.6（5番目の回答文を参照） |

#### B. デザイン・スタイリング実装

`design-template-1-guidelines.md` に従い、以下を実装する。

| 対象 | 実装内容 |
|-----|---------|
| 全体 | ベースカラー（白・淡いグレー背景）、キーカラー（`#0a4f9f`）を適用 |
| フォント | `system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif` |
| ヘッダー | シンプルなテキストナビ。現在のパイプ区切りをスタイリングで整える |
| ヒーロー | 2カラムグリッド（本文左・課題共感カード右）。CTAボタンをネイビー/白ボーダーで区分 |
| 実績セクション | グリッドカード（各項目を白カード・影付き） |
| サービス・事例・FAQ | カードグリッド（3列）。白背景・角丸・影 |
| CTA ボタン | メイン：ネイビー地・白文字・角丸大。セカンダリ：白地・ネイビー文字・ボーダー |
| フッター | 左右フレックス。コピーライト + サブナビリンク |

#### C. レスポンシブ対応

- ヒーロー・カードグリッド・実績グリッドは `max-width: 900px` 以下で1カラム縦積みに変更
- ヘッダーナビは縦積みに変更

## 実装方針

- CSS は `<style>` タグを `index.astro` と `Layout.astro` 内に直接記述（外部CSSファイルは Sprint 3 以降で整理）
- `design-template-1.html` の CSS をベースに、Astro のファイル構造に合わせて移植してよい
- セクション ID（`#hero`, `#stats`, `#services` 等）は Sprint 1 から引き継ぎ済み。ナビアンカーリンクとして活用可
