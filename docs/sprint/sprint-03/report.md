## v3 レビュー対応（①〜⑤）

### ① ヒーロー：3ゾーン構成への書き換え

**変更内容：**
- 従来の左右2カラム（`.hero-inner`）を廃止
- **上段 `.hero-top`**：`h1`（「そのExcel管理...」）をフル幅に配置。`padding: 56px 32px 32px`
- **下段 `.hero-bottom`**：リード文＋CTAボタン（左）と課題共感カード（右）の2カラムグリッド。`padding: 32px 32px 64px`
- 視線の流れ：パンチライン（フル幅）→ 本文＋CTA → 課題カード

**h1 フォントサイズ：** `3rem → 3.5rem`（PMレビュー承認済み）、SP時 `2.2rem`

---

### ② border-radius：フラット（0）

**変更内容：**
- `.hero` の `border-radius: 0 0 32px 32px` を `border-radius: 0` に変更
- ヒーローがページ上辺からフラットに開始し、下辺も直線で stats に続く
- B2B・専門サービスとしての「誠実・安定」の印象に一致

---

### ③ サービスカード：インラインSVGアイコン

各カード `<h3>` の直前に `<svg class="card-icon">` を挿入。stroke 色はブランドネイビー `#0a4f9f` で統一。

| サービス | アイコンモチーフ | 表現価値 |
|---------|--------------|---------|
| 業務システム開発 | モニター＋チェックマーク | 課題解決・効率化 |
| ツール・Webシステム開発 | 歯車（コグ） | 自動化・仕組み化 |
| 保守・運用サポート | シールド＋チェックマーク | 安心・継続性 |

絵文字不採用の理由：OS/ブラウザ依存・CSSコントロール不可・ブランドカラー統一不可。

### ⑤ pricing-box の視覚的差別化

**変更内容：**
- 背景：`#fff` + 影（カードと同系）→ `#eef3fb`（薄いブルー）+ 影なし
- 左ボーダー：`border-left: 4px solid #0a4f9f` を追加
- border-radius：`16px`（四辺）→ `0 12px 12px 0`（左辺フラット、右辺角丸）
- 見出し：通常の h3 スタイル → `font-size: 0.9rem; text-transform: uppercase; letter-spacing: 0.08em` で「ラベル」的な表現に

**設計意図：** 左アクセントボーダーにより「補足情報・参考値」のレイアウトとして識別しやすくなる。サービスカードと同列ではなく、付帯情報として視覚的に区別できる。

## PM確認事項

1. **`.section-inner` に `padding: 0 32px` を追加**  
   Sprint 3 初回で `.page` の padding に依存していたが、セクション背景色を将来的に全幅化する際にも対応できるよう、各セクションに内包する `.section-inner` に padding を移した。副作用はなし。

2. **ヒーロー `border-radius: 0` への変更**  
   stats セクションの背景（白）がヒーロー直下から始まるため、境界が直線になります。セクション間の境界線や余白の見え方をブラウザで確認してください。
