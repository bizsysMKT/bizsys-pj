# ユースケ用 Claude Code 起動ガイド

このファイルをClaude Codeに見せて「このプロジェクトの作業を手伝って」と伝えるだけでOK。

---

## このプロジェクトについて

ビズシス（漢那・畑野の2名体制のシステム開発会社）のHP・ブログ・SNSを構築する。  
ユースケが構築・初期運用を担当し、その後漢那に引き継ぐ予定。

**Claude Codeを使った構築・自動化の模索も目的の一つ。**

---

## ユースケがやること（5つ）

### ① HPの構築
- GitHub Pages + Astro + Formspree（問い合わせフォーム）で構築
- ドメイン：biz-sys.jp（設定は漢那が対応）

### ② ブログの構築
- Astro（HPと同リポジトリ）。記事はmarkdownで管理
- Claude Codeで下書き生成 → ユースケが確認・修正 → push → 自動デプロイ

### ③ X（旧Twitter）アカウントの作成と自動投稿設定
- アカウント名：@bizsys_jp（新規作成）
- ブログ記事公開 → 自動でX投稿（GitHub Actions）

### ④ 運用・自動化の仕組みづくり
- Claude Codeで記事下書きを生成する手順を整備
- 自動ビルド＆デプロイのパイプラインを作る
- **漢那が引き継げるように操作手順をドキュメント化する**
- 記事の公開頻度：週2〜3本

### ⑤ 効果測定・分析の仕組みづくり
- Google Analytics 4 を導入（アクセス数・流入経路・ページ別閲覧数）
- Google Search Console を導入（SEO・検索キーワード・表示回数）
- 月1回程度のレポートまとめの仕組みを整備する
