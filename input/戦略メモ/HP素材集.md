## 経歴・スキル（HP掲載用）

### 経歴
- システム開発・保守経験12年以上
- 通信業・製造業・建築業・教育業・小売業・美容業など多業種で経験
- 30社以上の企業・個人事業主のシステム開発・運用・保守に従事

### 対応範囲
- 業務分析 → 情報整理 → システム化の立案 → 設計 → 実装 → 試験 → 構築 → 運営まで**ワンストップ**

### 技術スタック
- 言語：Java / JavaScript / HTML / SQL
- FW：Spring Boot（5年）/ jQuery（10年）
- DB：PostgreSQL
- インフラ：Linux / Apache / Tomcat

## HP構成案

```
① トップ（ヒーローエリア）
   キャッチコピー ＋ 実績数字 ＋ 問い合わせCTA

② ビズシスとは
   2人の紹介・経歴・強み・ワンストップ対応の説明

③ サービス内容
   - 業務システム開発（30万〜100万円）
   - ツール・Webシステム開発
   - 保守・運用サポート（月1〜2万円）

④ 開発事例
   業種別のシステム事例カード（8種類）

⑤ ツール提供実績
   楽天・Yahoo出品ツール 351社導入など

⑥ よくある質問
   - 保守費用は必要ですか？
   - どんな相談でも受けてもらえますか？
   - 開発期間はどのくらいですか？
   - Excelからシステムに移行したいのですが相談できますか？

⑦ お問い合わせ
   Formspree フォーム
```
